Question 1(a) — 5 Marks
Without simplification, using only two-input basic gates, sketch the Logic Diagram for: F = (AB)A + C + (AB)Ā + B̄ + C̄   |   Also prove the universality of a NAND gate.
Part 1: Logic Diagram (using two-input gates only, no simplification)

We implement each term directly:

1 Compute AB → AND gate (A, B)
2 Compute NAND(A,B) = AB → NAND gate (A, B) — this gives (AB)̄
3 Compute T1 = (AB)·A → AND gate (AB, A)
4 Compute Ā → NOT(A)
5 Compute B̄ → NOT(B)
6 Compute C̄ → NOT(C)
7 Compute T2 = (AB̄)·Ā → AND gate (NAND(A,B), Ā)
8 Compute T3 = B̄ + C̄ → OR gate (B̄, C̄)
9 Compute T4 = T2 + T3 → OR gate (T2, T3)
10 Compute T5 = C + T4 → OR gate (C, T4)
11 Compute F = T1 + T5 → OR gate (T1, T5)
All multi-input ORs are chained as pairs of 2-input OR gates. All NOT gates are single-input. No simplification is performed.
Part 2: NAND Gate Universality

A NAND gate is universal because any Boolean function (NOT, AND, OR) can be built using only NAND gates.

NOT from NAND:
Connect both inputs together:
A NAND A = Ā
AND from NAND:
NAND output → another NAND (NOT):
NOT(A NAND B) = AB
OR from NAND: Apply De Morgan's theorem:
A + B = (A̅)̅ + (B̅)̅ = NOT(NOT A AND NOT B) = NAND(NAND(A,A), NAND(B,B))
i.e., invert both inputs using NAND, then NAND the results.
Conclusion
Since NOT, AND, and OR can all be implemented using NAND gates alone, and any Boolean function can be expressed using these three operations, the NAND gate is functionally complete (universal). The same applies to NOR gates.

Question 2(b) — 5 Marks
Simplify using Boolean Theorems:
(i) F = AB + (Ā + B̄ + C·C̄)̄   (ii) F = XY(X + XȲ) + Y(X + Y)
Part (i): F = AB + ( Ā + B̄ + C·C̄ )
1 C·C̄ = 0 (complement law)
2 Inside bar: Ā + B̄ + 0 = Ā + B̄
3 Ā + B̄ = A·B (De Morgan's: NOT of OR = AND of NOTs)
4 F = AB + AB = AB (idempotent)
F = AB
Part (ii): F = XY(X + XȲ) + Y(X + Y)
1 Inside first bracket: X + XȲ = X(1 + Ȳ) = X·1 = X (absorption)
2 XY(X) = X²Y = XY (idempotent: X·X = X)
3 Second term: Y(X + Y) = XY + Y·Y = XY + Y = Y(X + 1) = Y
4 F = XY + Y = Y(X + 1) = Y·1 = Y
F = Y

Subtract (447)₁₆ from (317)₁₀ using 2's complement. Verify in decimal.
Step 1: Convert both to decimal
1 (317)₁₀ = 317 (already decimal)
2 (447)₁₆ = 4×16² + 4×16 + 7 = 4×256 + 64 + 7 = 1024 + 64 + 7 = 1095
Step 2: Convert both to binary (10-bit for safety)

317 in binary: 256+32+16+8+4+1 = 0100111101 → 0100111101

Verify: 256+32+16+8+4+1 = 317 ✓

1095 in binary: 1024+64+4+2+1 = 10001000111 → needs 11 bits: 10001000111

Step 3: Operation — 317 − 1095

We compute 317 − 1095 = −778 (result will be negative)

Use 12-bit representation:

317 = 000100111101

1095 = 010001000111

3 1's complement of 1095: flip all bits of 010001000111 → 101110111000

Decoder-based Implementation (3-to-8 decoder)

A 3-to-8 decoder with inputs A, B, Bin generates all minterms m0–m7 as outputs (active-low or active-high depending on type). Use active-high outputs with OR gates:

  • 3-to-8 Decoder inputs: A, B, Bin
  • Decoder outputs: m0=000, m1=001, m2=010, m3=011, m4=100, m5=101, m6=110, m7=111
  • D = OR(m1, m2, m4, m7) — connect those 4 decoder outputs to a 4-input OR
  • Bout = OR(m1, m2, m3, m7) — connect those 4 decoder outputs to a 4-input OR

This is the standard method: decoder generates all minterms, OR gates select the required ones.

Question 6(b) — 5 Marks
Explain the working principles of all types of flip-flops with figures and truth tables.
1. SR (Set-Reset) Flip-Flop

Basic bistable element. S=1 sets output Q=1. R=1 resets Q=0. Both=1 is forbidden.

SRQ(next)Q'(next)Comment
00QQ'No change
0101Reset
1010Set
11??FORBIDDEN
Built using two cross-coupled NOR gates (or NAND gates for active-low). Characteristic equation: Q(n+1) = S + R'Q
2. JK Flip-Flop

Improved SR FF. Eliminates forbidden state: when J=K=1, output toggles.

JKQ(next)Action
00QNo change
010Reset
101Set
11Q'Toggle
Characteristic equation: Q(n+1) = JQ' + K'Q. Built with SR FF adding feedback from Q to R-input gate and Q' to S-input gate. Clock-triggered (edge or level).
3. D (Data) Flip-Flop

Single input D. Output follows D on clock edge. Eliminates invalid state of SR FF.

DQ(next)Action
00Reset
11Set
Characteristic equation: Q(n+1) = D. Built by connecting D to S and D' to R of an SR FF. Used as a 1-bit memory/register. Most commonly used FF in digital systems.
4. T (Toggle) Flip-Flop

Single input T. When T=1, output toggles on each clock edge. When T=0, holds state.

TQ(next)Action
0QNo change
1Q'Toggle
Characteristic equation: Q(n+1) = T⊕Q. Built from JK FF with J=K=T. Used in counters and frequency dividers (each T FF divides frequency by 2).
Comparison Summary
TypeInputsCharacteristic EquationMain Use
SRS, RQ+ = S + R'Q (S·R=0)Basic memory element
JKJ, KQ+ = JQ' + K'QUniversal FF, counters
DDQ+ = DRegisters, data storage

All flip-flops are clock-triggered (synchronous). Edge-triggered (positive/negative edge) FFs are more common in modern digital design than level-triggered latches.
